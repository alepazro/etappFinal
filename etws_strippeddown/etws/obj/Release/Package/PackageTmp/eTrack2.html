Introduction

  • eTrack Pilot API is a set of RESTful services that support the mobile eTrack Pilot application for Android tablets and smartphones.
  • Methods of the eTrack Pilot API use JSON on the requests and the responses. Some methods do GET requests with in-line params (query string).
  • You will need valid credentials (Login and Password) to use the eTrack Pilot API. With valid credentials, the API will return an authorization token that must be used to invoke every method of the API. This authorization token has an expiration date. When the token expires, you must get a new one by providing your credentials.
  • eTrack Pilot API can be used from any system that supports RESTful services (most modern development tools do). You can do HTTP Requests via AJAX or via network libraries from your system to connect to eTrack Pilot API and get/set information related to your eTrack account.

Enter your credentials to retrieve your authorization token

Credentials have to be requested to your Account Manager.

The method to get the authorization token is: 

Method: https://etrack.ws/pilot.svc/authorization
Type: GET
Request: ?login={login}&password={password}&expDays={expDays}&lat={latitude}&lng={longitude}
Response: {isOk:/true//false/, msg:/message/, fullName:/fullName/, isCheckedIn: /true//false/, checkedInSince:/date/, token:/token/, deviceId: /deviceId/}

The method to get the authorization token via Authorization2 is: 

Method: https://etrack.ws/pilot.svc/authorization2
Type: GET
Request: ?login={login}&password={password}&lat={latitude}&lng={longitude}
Response: {isOk:/true//false/, msg:/message/, companyId:/companyId/, userId:/userId/,fullName:/fullName/, token:/token/, deviceId: /deviceId/}

To start testing the API, please enter your credentials below to retrieve your authorization token.

Recover Credentials

The API offers a method to request the credentials via email. The user has to enter a valid email. This email will be sent to the backend system. If the email exists and belongs to an active account, the credentials will be sent to that email.

The method to recover credentials is: 

Method: https://etrack.ws/pilot.svc/recoverCredentials
Type: POST
Request: {email:/email/}
Response: {isOk:/true//false/}

Type: POST
Request: {token: /token/, jobId:/jobId/, imgType:/imgType/, imgId:/imgId/, imgName:/imgName/, fileName:/fileName/, fileType:/fileType/, imgData: /imgData/, eventDate: /eventDate/, lat:/latitude/, lng:/longitude/}
